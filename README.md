# cryptocurrency-market-dashboard-excel
Cryptocurrency Market Dashboard built using Microsoft Excel. This project analyzes crypto market data using pivot tables, charts, and interactive visualizations.

# Cryptocurrency Market Dashboard (Excel Project)

## Project Overview

This project presents a **Cryptocurrency Market Analysis Dashboard** created using **Microsoft Excel**.
The goal of the project is to analyze cryptocurrency market data and visualize important market insights through an interactive dashboard.

The dataset contains **1000 cryptocurrency records** with various financial and technical indicators such as market capitalization, price changes, trading volume, supply metrics, and community engagement scores.

---

# Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Data Visualization

---

# Dataset Description

The dataset includes **25 columns** related to cryptocurrency performance and analytics.

### Main Columns

| Column Name           | Description                              |
| --------------------- | ---------------------------------------- |
| Date                  | Date of record                           |
| Month                 | Month of the record                      |
| Crypto_Name           | Name of the cryptocurrency               |
| Symbol                | Crypto symbol (BTC, ETH etc.)            |
| Market_Cap_USD        | Total market capitalization              |
| Price_USD             | Current price in USD                     |
| Volume_24h_USD        | 24-hour trading volume                   |
| Circulating_Supply    | Coins currently in circulation           |
| Total_Supply          | Total coins available                    |
| Max_Supply            | Maximum possible coins                   |
| Market_Dominance_%    | Market share percentage                  |
| Change_1h_%           | Price change in 1 hour                   |
| Change_24h_%          | Price change in 24 hours                 |
| Change_7d_%           | Price change in 7 days                   |
| Rank                  | Cryptocurrency ranking                   |
| Category              | Type of crypto (DeFi, NFT, Payment etc.) |
| Exchange              | Exchange platform                        |
| Country               | Origin country                           |
| Launch_Year           | Year of launch                           |
| Mining_Type           | Mining mechanism                         |
| Transaction_Speed_TPS | Transactions per second                  |
| Energy_Usage_Index    | Energy consumption index                 |
| Adoption_Score        | Global adoption level                    |
| Developer_Activity    | Developer contribution score             |
| Community_Score       | Community engagement score               |

# Project Structure

### 1️⃣ Data Sheet

Contains the **raw cryptocurrency dataset (1000 records)** used for analysis.

### 2️⃣ Tables Sheet

Pivot tables were created from the dataset to summarize and analyze key metrics such as:

* Market capitalization
* Trading volume
* Cryptocurrency ranking
* Adoption score

### 3️⃣ Final Dashboard

The final Excel dashboard visualizes important insights using charts such as:

* Market Cap Analysis
* Cryptocurrency Price Comparison
* Volume Analysis
* Adoption & Community Score
* Market Dominance Distribution

# Key Insights

The dashboard helps analyze:

* Top cryptocurrencies by **market capitalization**
* Trading activity using **24h volume**
